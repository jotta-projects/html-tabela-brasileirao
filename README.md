# ⚽ Tabela Brasileirão

Site desenvolvido em HTML e CSS sobre os times do Campeonato Brasileiro.

## 📋 Sobre o projeto

Projeto desenvolvido na ETEC durante o curso de Desenvolvimento de Sistemas. O site apresenta informações e imagens dos principais times do Brasileirão.

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**

## 🚀 Como visualizar

Basta abrir o arquivo `index.html` no navegador — não precisa instalar nada!
